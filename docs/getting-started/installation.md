# Installation

## Requirements

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) package manager

## Install with uv

```bash
uv tool install crow-cli --python 3.14
```

## Verify Installation

```bash
crow-cli --version
```

## Development Install

For contributing or development:

```bash
git clone https://github.com/crow-ai/crow-cli
cd crow-cli
uv sync
```
